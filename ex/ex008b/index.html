Código-fonte / Pré-formatação

O comando document.getElementById('teste') é escrito em linguagem javaScript.
A tag code ela transforma um texto em mono-espaçada, assim facilitando na lida do código.
Outro exemplo: 

            
    num = int(input('Digite um número: '))
    if num % 2 == 0:
        print(f'O múmero {num} é par.')
    else:
        print(f'O número {num} não é par.')
    print('Fim do programa')

com a tag <code> e <pre> juntas podemos transformar um texto em formato de código. A tag <pre> considera tudo o que você digitou dentro dela.

Citações

Com a tag <q> podemos colocar aspas nas frases, principalmente com o sentido de citação.
Exemplo: Como diria o pai de um amigo: O computador é um burro muito rápido

Citações completas

Com a tag <blockquote> podemos utilizar citações em formato de bloco.
Exemplo utilizando lorem:

Lorem ipsum dolor, sit amet consectetur adipisicing elit. Numquam ducimus magni laboriosam sit suscipit amet error, voluptatum eveniet quam consectetur saepe delectus, atque provident? Sequi, laboriosam. Culpa ullam ipsum perferendis.

E com o parametro cite para dizer o que você está citando, não muda nada na página principal, mas isso ajuda nos mecanismos de busca e também deixa o seu código HTML mais organizado, a parte interna do seu site vai saber o que você está citando, mantem os direitos daqueles que produzem conteúdo.

Abreviação
