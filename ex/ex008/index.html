Principais formatações

Negrito

Podemos utilizar a tag <b> para botar em negrito, mas ela não é semântica.

Já a tag <strong> é semântica, tem significado de dar um em destaque .

Itálico / Ênfase

Podemos utilizar a tag <i>, mas não é semântica. O certo é utilizar as tags semânticas, pois tem um significado.
A tag semântica para ítalico é <em>, que tem o significado de ênfase

Texto marcado

Para marcar um trexo de um texto ou palavra na página, utiliza-se a tag <mark>.

Texto grande e pequeno

Existia a tag big mas ela se tornou obsoleta, mas a tag <small> não.

Texto deletado

Podemos marcar um texto como deletado com a tag <del> para dizer que deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado com a tag <ins>

Texto sobrescrito

Para inserir algo acima de uma palavra, podemos utilizar a tag <sup>.
Exemplo: x 2, 2 x, qualquer coisa qualquer coisa

Texto sobrescrito

Já para inserir coisas em baixo, pode utilizar a tag <sub>
Exemplo: H 2O, n 1, qualquer coisa qualquer coisa
