Parágrafos, emojis, quebra de linha e caracteres especiais

Você pode escrever um parágrafo de qualquer jeito. Basta colocar tudo no meio do par de tags <p> e </p>
Você também pode usar a tag <br> que significa "break row" para quebrar linha

Existem caracteres especiais que geralmente usamos & e no final ; para usar
Exemplo: ∑, δ, ©, ∩, Δ, ∇, ®, ↕.

Para adicionar emojis usamos codigos específicos, pode usar &#x para dizer que o valor é hexadecimal para iniciar um emoji, pode usar o emojipedia para pegar o código.
Exemplo: 🖖 🤓
