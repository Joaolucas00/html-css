Modificando tags box-level

Vamos ver alguns atributos para tags box level. Border width é a largura da borda, border color é a cor da borda, border style é o estilo da border. Assim como nos outros podemos modificar cada lado e atributo das caixas, pode utilizar o shorthand para isso, mas senão, coloque os atributos de modificação da caixa em sentido horário. Também tem como transformar uma tag box-level em uma tag inline-level com o atributo display, display inline ou display block, também tem o inline-block que é um inline com algumas características de box-level como altura e largura.
