Usando links

links externos

Links externos são links que apontam para outro site, utilizando a tag <a> que significa "âncora", usando um parametro obrigatório "Href" para botar a URL.
Exemplo: Acesse o materia de apoio em pdf

Sem o atributo "target", após clicar o link, vai começar uma nova navegação, sobrepondo o seu site. Por isso podemos usar o atributo "target" com os seguintes valores:

_blank
vai abrir o link em uma nova janela em blanco
_self
Vai abrir o link na janela ou frame atual (padrão)
_top
Vai desfazer todos os frames e abrir o destino no navegador completo
_parent
similar ao uso do _top em uma referência à janela mãe. nome-do-frame caso esteja usando frames, indicar o nome da janela a abrir

Exemplos

  • Usando _blank: Acesse o materia de apoio em pdf
  • Usando _self: Acesse o materia de apoio em pdf
  • Usando _top: Acesse o materia de apoio em pdf
  • Usando _parent: Acesse o materia de apoio em pdf

Agora no HTML5 você pode usar o parametro rel para indicar o tipo de link.

Links internos

São links que referencia a outra página que está dentro do servidor, que diz respeito a sua própria página. Index é por onde o site é aberto.
Exemplo: Segunda página
Terceira página (só que em outra pasta)

Não existe um rel "internal", mas existe "next" indicando para a próxima página e "prev" indicando para a página anterior.

Nofollow

Tem ocasiões em que você quer linkar para outro site, mas você não quer transferir nenhum relevância pra ele. Ou seja, sem indexação pelo Google. Exemplo: quando você quiser apontar algo como um mau exemplo na Internet. Google recomenda usar para links patrocinados, para que o mecanismo de busca não perpetue esta busca.
