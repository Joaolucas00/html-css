Download

Podemos também fazer donwload de arquivos com links. Usando o parametro download e indicando o nome do arquivo, e para baixa utilize a tag href e dentro dela indique o arquivo a ser instalado. Configurando o type para o tipo de arquivo.
Exemplo: Baixe o pdf do capítulo 10

Mas como eu sei qual type utilizar para o tipo de arquivo? Você pode acessar o site iana.org, há uma lista de todos os media type. https://www.iana.org/assignments/media-types/media-types.xhtml
