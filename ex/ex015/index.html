id

Para diferenciar um h1 de outro h1 e assim nós fazermos formatações diferentes entre os h1 (qualquer tag) nós podemos criar um id (indentificador) para indentificar eles, se quiser nem precisa colocar a tag, basta adicionar # e o nome que você colocou no id. Utilize # ao lado da tag para poder personalizar dentro das CSS. Tudo que é em id em CSS é #, HTML id = # CSS

class

Caso você queira utilizar a mesma configuração para mais de um elemento, utilize class. Para indentificador o class no CSS uiliza um pontinho(.). No HTML é class, em CSS é ., HTML class = .CSS. Tem como colocar mais de uma class na mesma tag, e com id também, mas na mesma tag só pode um id.

span

span é um tag que não altera em nada, só ajuda a você pegar um trexo no texto para modificar.
