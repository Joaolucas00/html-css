Um pouco sobre div

Div é uma pequena área que vai ocupar um determinado espaço, por padrão vai ocupar da ponta a outra, serve apenas para demarcar um espaço qualquer dentro da nosso site. Tem alguns atributos como height(altura), width (largura), border(borda).

>

Você pode criar um seletor especial utilizando >, significa algo está dentro de, exemplo: div > p (significa, p que está dentro de, filha de, uma div), ele só vai mexer num parágrafo que está dentro da div. Podemos utilizar também com pseudo-classes. Tem como utilizar mais de uma vez como por exemplo: a > div#principal > p, assim vai apenas modificar o parágrafo que está dentro de uma div com o id de "principal", que por sua vez está dentro de uma âncora(link). Dentro do CSS chamamos isso > de children

Active

É um estado de elemento que é ativada quando uma unidade de elemento for ativada, como por exmplo após ser clicada ou segurada.

Pseudo-classes

Uma pseudo-classe é uma palavra-chave adicionado às declarações de um seletor após um sinal de dois pontos e especifica um estado especial de um elemento. Existem várias pseudo-classe, como: :hover, :visited, :active, :checked, :empty e :focus. Clique para ver um exemplo!

Pseudo-elementos

O pseudo-elemento pode agir nas classes, nos elementos, nos id. ELe mexe no conteúdo, diretamente no conteúdo, periférico. Utilizando :: . Alguns exemplos de pseudo-elementos: ::before, ::after, ::first-letter, ::first-line.
