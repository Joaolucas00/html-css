Alinhamento CSS

Text-align

Para usar alinhamento com CSS é bem simples, é só usar o text-align. Existe outra propriedade que é o text indent, ele dá um espaço no começo do parágrafo. O text-align vem com os parametros center(centro), left(esquerda), right(direita), end(final), start(), justify(alinhamento em ambos os lados )
