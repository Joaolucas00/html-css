Fontes em CSS

Podemos alterar a fonte da palavra com font-family no CSS. É bom colocar mais de uma fonte no font-family porque alguns dispositivos não são compatíveis com certas fontes. Para saber quais fontes são seguras acesse esse site., sempre termina a sua safe font com sans-serif(ou outro, ex: monospace) porque todos os dispositivos vão interpretar e vão usar alguma fonte generica sem serifa.

Tamanho da fonte

Há várias medidas que podemos usar para o tamanho da fonte, como cm(centímetros), in(polegadas), pt(pontos), pc(paicas), px(pixels), etc. A W3C recomenda o uso do px ou do em. Também temos medidas absolutas e medidas relativas. 16px é o tamanho normal da fonte que também é igual a 1em.

  • Meidadas Absolutas
    1. cm(centímetro)
    2. mm(milímetro)
    3. in(polegadas)
    4. pt(pontos)
    5. px(pixels)
    6. pc(paicas)
  • Medidas Relativas
    1. em(relativo ao tamanho original da fonte, geralmente é igual a 16px)
    2. ex(relativo ao x minúsculo)
    3. rem(relativo a configuração da fonte no body)
    4. vw(relativo ao viewport, tamanho de tela especialmente largura)
    5. vh(relativo a altura da viewport)
    6. %

É importante usar o em porque ele é relativo ao tamanho da fonte já que algumas telas tem tamanhos difertentes.

Peso da fonte

Cuidado para não confundir! As palavras "weight", "width" e "height" são parecidas, mas tem significados diferentes. weight é peso, width é largura e height é altura. Utilizando o font-weight podemos alterar o peso da fonte, tem como padrão os parâmetros lighter, normal, bold e bolder ou você pode usar um parâmetro númerico que vai de 100 a 900.

Com font-style podemos alterar o estilo da fonte, para ítalico por exemplo. Se quiser um texto sublinhado use text-decoration. Algumas fonte não tem essas configurações então tome cuidado.

shorthand font

Para não ficar colocando todas essas configurações em cada linha use "font:" e adiciona nessa ordem:
font-style -> font-weight -> font-size -> font-family
