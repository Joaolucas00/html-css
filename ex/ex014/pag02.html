Fontes externas

As vezes a fonte que você procura não vai estar no google fonte, então vai ter que procurar em outro lugar. Você pode acessar o site DaFont para adquirir outros tipos de fontes.

OTF+e |TTF

Para adicionar um arquivo fonte TTF ou OTF, utilize 


    @font-face {
        font-family: ;
        src: url('CELTG___.TTF') format('opentype');
    }

Sobre Alinhamento

Para tipos de format temos opentype(otf), truetype(ttf), embedded-opentype, truetype-aat (Apple Advanced Typography) e svg. No font family você vai nomear o nome da fonte para utilizar no CSS. Cada navegador vai ter ou não compatibilidade com esses tipos de arquivo, então é recomendável que utilize os dois arquivos juntos, geralmente funciona em todos os navegadores.
