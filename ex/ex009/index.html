Trabalhando com listas

Listas Ordenadas

Nas litas ordenadas a ordem faz diferença. Com a tag <ol> podemos criar listas ordenadas, utilizando junto também a tag <li> para criar cada elemento da lista (o fechamento da tag li é opcional).
Exemplo:

  1. Acordar
  2. Tomar café
  3. Escovar os dentes

Podemos também alterar a formatação do índice do item, como por exemplo, em vez de ser números comuns, podemos utilizar números romanos ou letras com o parametro type que oferece 5 opções, e também podemos indicar o início da lista com o parametro start.
Exemplos:

  1. Feijão
  2. Arroz
  3. Carne
  4. Batatas

Listas não Ordenadas

Listas não ordenadas, como o próprio nome diz, são listas onde não há a necessidade de ordenalas. Utilizamos a tag ul para criar esses tipos de listas, com a mesma tag de antes li para criar elementos e type para personalizar.

Juntando listas

Listas também podem estar dentro de outras listas.
Exemplo:

    Liguagens de programação
  1. Antigas
    1. CLipper
    2. Visual Basic
    3. Fortran
    4. Delphi
  2. Novas
    1. PHP
    2. Python
    3. JavaScript
    4. Kotlin

Você pode fazer quantos niveis você quiser

Lista de Definição

Lista de definição é como se fosse um dicionário. Utilizando a tag dl para criar, dt para ditar o termo e dd para a definição
Exemplo:

HTML
Liguagem de marcação para a criação do conteúdo de um site.
CSS
Linguagem de marcação para a criação do design de um site.
JavaScript
Linguagem de programação para a criação de interatividade de um site.
