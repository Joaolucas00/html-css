Imagens dinámicas

tag picture

Com a tag picture, podemos tratar imagens, como por exemplo, se for um celular que está acessando seu site, ele muda a imagem de 1000px para 300px, ou se for um tablet para 600px, isso tudo apenas utilizando o tag source dentro de picture.

A ordem da tag source dentro de picture deve ser seguida, se não, não funciona. A ordem é de cima do picture para baixo e a img na ultíma linha, as mais pequenas sempre vem em cima das mais grandes.
