CSS

CSS é um mecanismo para adicionar estilos no documento HTML, é a parte de design do site. É onde a gente vai colocar cor, fonte, tamanho, etc. Neste capítulo abordaremos as três formas de usar o CSS.

CSS inline

É a forma mais simples para aplicar os estilos em áreas pontuais em nosso site, que é utilizando o CSS dentro de parâmetros do HTML. Esta maneira também tem suas limitações, se por exemplo você mudar a cor de um h1, não vai mudar a cor de outro h1, e também vai poluir o seu documento HTML. Esse método é para configurações pontuais, em partes específicas no documento HTML.
