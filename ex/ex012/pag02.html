CSS internal style

Estilos internos, ou locais, são os estilos que estão separados da área body, do conteúdo do site (HTML). Utilizando a tag style dentro da área head, tudo que for colocado dentro da tag style vai ser considerado CSS, esse estilo é baseado em seletores, não vai alterar um ponto específico do site, mas o todo. Esse método melhor que o CSS inline, já que você separa o conteúdo do site (HTML) da parte do design do site (CSS). Mas ainda não é o melhor jeito, o certo é manter o CSS separado do documento HTML.
