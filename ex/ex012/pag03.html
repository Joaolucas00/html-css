CSS external

Estilo externo, é o CSS aplicado só que em um arquivo separado do documento HTML. Isso é bom para manter tudo organizado, separando a perte de design da parte de conteúdo. Use a tag link dentro de head para linkar o arquivo CSS como padrão de design. (você pode usar mais de um link de CSS no mesmo documento HTML)
