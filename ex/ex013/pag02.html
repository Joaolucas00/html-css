Gradiente

Podenos usar o gradinte utilizando o CSS com "background-image: linear-gradient(to left (ou right, top, bottom, ou em graus dag), white, blue)"" e ficar com um visual bacana no site.

Configuração global CSS

Utilize "*" e "{" para criar uma configuração global e assim você configurar tamanho, largura , etc. Também dá pra usar o degradê em formatos usando "radial-gradient" com background-image. Exemplo: background-image: radial-gradient(circle, #FF810E, #FDAA02, #F465BD, #ED6980, #FF025F);
